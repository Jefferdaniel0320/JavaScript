# CURSO JAVASCRIPT MODERNO

Se recomienda colocar la etiqueta **script** despues de finalizar el div: 
~~~
</div>
<script>
        alert('Hola Mundo!');
</script>
~~~
Pero lo mas aconsejable es usar un archivo aparte desde una carpeta:
~~~
<script src="js/app.js"></script>
~~~

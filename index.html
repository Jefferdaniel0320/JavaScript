Página Principal de Proyectos y Ejercicios
JavaScript

En esta página, descubrirás ejemplos en JavaScript que aplican estructuras de control, iteradores, métodos de arrays, manipulación del DOM, manejo de eventos, módulos, APIs y muchas otras cosas. Cada sección contiene ejercicios prácticos para reforzar tus habilidades.

Ejemplos

Ejemplo 1

Carrito de Compras

En esta sección, partimos de una página existente en HTML y CSS. Luego, implementamos eventos para agregar cursos al carrito de compras, aumentar la cantidad y eliminarlos utilizando JavaScript.

Ejemplo 2

Simulador Envió de Mail

En esta sección, partimos de una página existente en HTML y CSS. Luego, implementamos eventos para validar cada uno de los campos del formulario con el fin de habilitar los botones de "Enviar" y "Reset". Además, simulamos el envío de un correo electrónico.
